Glyphive E2E benchmark fixture note-001.
This is a small, deterministic, synthetic document used only to exercise the
create -> rasterize -> OCR -> extract round trip. It is intentionally boring:
plain ASCII prose repeated with minor variation so the payload compresses
moderately and is long enough to give Reed-Solomon a real multi-line parity
budget, matching how the format is meant to be used in practice.
